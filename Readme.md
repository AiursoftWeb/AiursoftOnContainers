# Aiursoft On Containers

One project to run all Aiursoft micro-services

## How to run

```bash
$ docker-compose build
$ docker-compose up
```

## How to develope

We strongly suggest using `Visual Studio Code` to edit this project. Instead, you can use Visual Studio to run it.
